# Launch A Tab Using Current Token (LATUCT)

## Installation Instructions (Edge or Chrome)

1. `git clone https://github.com/wavemotionio/latuct.git`
1. Open `edge://extensions/` or `chrome://extensions/`
1. Enable developer mode
